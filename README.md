## What this project does

Overwatch 2 Battle Pass Calculator is a small web-based utility that helps you check battle pass progress without having to work out the numbers manually. It centers on the details most players care about: current standing, progress speed, and the remaining distance to the end of the pass.

The tool is aimed at players who want a fast browser-based progress snapshot with tier-aware tracking for both base tiers and prestige tiers. It also estimates completion using either your present pace or a target pace, while including coin earnings so you can see progression more clearly.
